## [dirtyfrag](https://github.com/V4bel/dirtyfrag) #1

> 通过内存碎片与 `Dirty Pipe` 类漏洞结合，实现内核级权限提升。该项目用 C 语言演示了一种新型攻击向量，引发安全社区高度关注。

<p><img src="https://opengraph.githubassets.com/1/V4bel/dirtyfrag" alt="dirtyfrag" style="max-width:100%;border-radius:8px;"></p>

传统的漏洞利用往往依赖堆栈溢出或野指针，而 **dirtyfrag** 另辟蹊径，利用内存碎片化状态绕过内核防护。作者 **V4bel** 在 Linux 内核环境复现了这一攻击。

技术核心在于构造特定大小的内存请求，使缓存页与用户态页重叠，进而触发 **page-cache** 写传播。项目包含完整的 PoC 代码和执行脚本。

发布后仅一周即获得 **619 Stars**、**74 Forks**，成为近期最热的内核安全研究项目。安全研究员指出该方法对内核 5.8 以下版本威胁较大。

目前项目处于早期公布阶段，未提供补丁建议。适合内核开发者和安全研究员逆向学习，也提醒用户及时更新内核版本。

相关链接：
- 项目地址：[V4bel/dirtyfrag](https://github.com/V4bel/dirtyfrag)

## [LifeManga](https://github.com/iam567/LifeManga) #2

> 一款用 Swift 编写的开源漫画阅读器，支持本地导入与在线源抓取，界面高度可定制。适合 iOS 端漫画爱好者。

<p><img src="https://opengraph.githubassets.com/1/iam567/LifeManga" alt="LifeManga" style="max-width:100%;border-radius:8px;"></p>

iOS 平台漫画阅读器长期被商业应用垄断，**LifeManga** 以开源形式提供无广告、零跟踪的阅读体验。开发者 **iam567** 表示初衷是“为阅读本身而设计”。

核心功能包括：双击缩放、自动切割双页、夜间模式，以及通过自定义规则解析各类漫画网站。支持导出阅读进度 JSON。

项目在 GitHub 收获 **146 Stars**、**36 Forks**，社区贡献了多个源解析器。测试覆盖 iOS 15~18 系统。

对比同类项目，LifeManga 更注重隐私——所有网络请求可审计，不内置任何遥测 SDK。未来计划加入 iCloud 同步。

相关链接：
- 项目地址：[iam567/LifeManga](https://github.com/iam567/LifeManga)

## [blooket](https://github.com/Binarylunanotice/blooket) #3

> 针对教育游戏平台 Blooket 的自动化脚本，可一键完成刷分、改昵称、挂机等操作。用 JavaScript 实现，直接运行于浏览器控制台。

<p><img src="https://opengraph.githubassets.com/1/Binarylunanotice/blooket" alt="blooket" style="max-width:100%;border-radius:8px;"></p>

**Blooket** 是北美流行的课堂互动游戏，学生通过答题获取虚拟奖励。该脚本利用未鉴权的 API 接口实现自动答题与代币刷取。

代码封装成独立函数，调用 `run()` 即可启动。支持选择游戏模式（黄金、大逃杀等），并模拟键盘事件绕过简单反爬。

当前获 **23 Stars**，尚无 Fork。开发者明确声明“仅供教育测试”，但平台方可能视为违规行为。建议仅在个人沙箱环境体验。

项目仅包含一个 `script.js` 文件，结构简单。同类工具有 `blooket-hacks`，但该脚本的自动化程度更高。

相关链接：
- 项目地址：[Binarylunanotice/blooket](https://github.com/Binarylunanotice/blooket)

## [Awesome-Sparse-Attention-Video-Diffusion](https://github.com/Mutual-Luo/Awesome-Sparse-Attention-Video-Diffusion) #4

> 系统收集了稀疏注意力机制在视频扩散模型中的最新论文、代码和资源。面向视频生成研究者，是该领域第一个专题列表。

<p><img src="https://opengraph.githubassets.com/1/Mutual-Luo/Awesome-Sparse-Attention-Video-Diffusion" alt="Awesome-Sparse-Attention-Video-Diffusion" style="max-width:100%;border-radius:8px;"></p>

视频扩散模型计算开销巨大，完整注意力机制难以处理长时序列。**稀疏注意力** 通过仅关注关键帧或局部窗口，大幅降低显存占用。

列表收录了 **CogVideo**、**VideoFusion**、**MagicVideo** 等代表性工作的实现，并附有官方链接。按学术论文、代码仓库、博客进行三级分类。

当前获 **20 Stars**、**1 Fork**，虽数据不大，但内容质量较高。每个条目都包含引用格式，方便直接写入论文相关工作部分。

相比普通 `awesome-list`，该项目还额外整理了参数量、推理速度对比表。适合准备进入视频生成方向的研究人员作为入门索引。

相关链接：
- 项目地址：[Mutual-Luo/Awesome-Sparse-Attention-Video-Diffusion](https://github.com/Mutual-Luo/Awesome-Sparse-Attention-Video-Diffusion)

## [note-slides](https://github.com/gainubi/note-slides) #5

> 将 Markdown 笔记自动渲染为 HTML 幻灯片，支持分页、代码高亮与主题切换。基于 Python 命令行工具，无需浏览器插件。

<p><img src="https://opengraph.githubassets.com/1/gainubi/note-slides" alt="note-slides" style="max-width:100%;border-radius:8px;"></p>

许多知识工作者习惯用 Markdown 做学习笔记，但演示场景下仍需要手动复制到 PPT。**note-slides** 只需要一行命令 `noteslides input.md` 即可生成幻灯片。

核心技术依赖 `markdown-it-py` 解析器，配合 `reveal.js` 作为渲染引擎。默认支持分段动画、演讲者备注和导出 PDF。

GitHub 上获 **17 Stars**、**2 Forks**，社区提出需求支持 LaTeX 公式与图表嵌入。作者计划加入 `--template` 参数实现自定义样式。

与 `marp-cli` 相比，note-slides 更轻量，无需安装 `chromium` 即可工作。特别适合快速将读书笔记转为分享材料。

相关链接：
- 项目地址：[gainubi/note-slides](https://github.com/gainubi/note-slides)

## [Live-YT-Translator](https://github.com/petergpt/Live-YT-Translator) #6

> 一个 JavaScript 书签脚本，可在 YouTube 直播页面实时将英文 CC 字幕翻译成多语言 overlay 字幕。利用浏览器内置翻译 API。

<p><img src="https://opengraph.githubassets.com/1/petergpt/Live-YT-Translator" alt="Live-YT-Translator" style="max-width:100%;border-radius:8px;"></p>

跨语言实时直播一直是痛点，第三方服务往往需要付费或复杂的转接。**Live-YT-Translator** 仅需浏览器运行，不发送任何数据到第三方服务器。

技术原理：监听页面的 `TextTrack` 变化，通过 `window.TransApi`（需要用户提供 API Key）逐句翻译，然后注入到视频上方 `<div>` 中。

项目获 **16 Stars**，代码行数不到 300，结构清晰。兼容 Chrome、Firefox 与 Edge。

目前仅支持英→中、日、韩、法四种语言。作者提醒直播延迟约 2~3 秒，且依赖 `TransApi` 的免费配额。

相关链接：
- 项目地址：[petergpt/Live-YT-Translator](https://github.com/petergpt/Live-YT-Translator)

## [WIfikeyXposed](https://github.com/TYOPXN360/WIfikeyXposed) #7

> 基于 Xposed 框架的模块，用于提取安卓系统中已保存 WiFi 密码。需要 Root 权限及 Xposed 环境。

<p><img src="https://opengraph.githubassets.com/1/TYOPXN360/WIfikeyXposed" alt="WIfikeyXposed" style="max-width:100%;border-radius:8px;"></p>

忘记 WiFi 密码又想分享给其他设备？**WIfikeyXposed** 通过 Hook 系统 `WifiManager` 的 `getWifiConfiguration` 方法，直接读取明文密码。

模块体积仅 20KB ，安装后在 Xposed Installer 中激活即可。支持输出二维码分享。

当前 **7 Stars**，代码量较小，仅含一个 Java 类。适用于 Android 8~11 系统（12 以上需额外适配）。

需要注意的是，此模块会触发系统安全警告，且可能被杀毒软件标记。建议仅在自己设备上进行密码找回。

相关链接：
- 项目地址：[TYOPXN360/WIfikeyXposed](https://github.com/TYOPXN360/WIfikeyXposed)
