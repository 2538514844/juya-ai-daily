# 2025年4月17日 GitHub 开源项目精选

## 概览
本期精选涵盖安全扫描、AI 图像生成、浏览器广告拦截、文件工具及开发辅助等小而美的项目，虽然整体 Stars 不高，但每个项目都聚焦于特定场景的实用问题，值得关注。

## 安全工具
- **[assetnote/cpanel2shell-scanner](https://github.com/assetnote/cpanel2shell-scanner)** ⭐ 50 — 高精度扫描 CVE-2026-41940（cPanel & WHM 认证绕过漏洞），帮助管理员快速检测受影响环境。

## AI / 大模型
- **[therichardngai-code/gpt-image-2-pro-max](https://github.com/therichardngai-code/gpt-image-2-pro-max)** ⭐ 66 — 基于 GPT 模型的图像生成与编辑进阶工具，旨在提供更高质量、更可控的图像输出。

## 开发工具与效率
- **[openclaw/crabbox](https://github.com/openclaw/crabbox)** ⭐ 67 — 实现“预热容器、同步差异、运行测试套件”的开发辅助工具，帮助团队在 CI/CD 中加速测试与部署流程。
- **[AKYMYK/FileToolkit](https://github.com/AKYMYK/FileToolkit)** ⭐ 51 — 提供一系列常用文件操作工具，简化批量重命名、格式转换等日常处理任务。
- **[electricant/adblock-rust-manager](https://github.com/electricant/adblock-rust-manager)** ⭐ 48 — 一款 Firefox 扩展，用于启用和管理 Firefox 内置的 adblock-rust 广告拦截引擎，让用户无需安装额外扩展即可获得原生拦截能力。
- **[betta-tech/ejemplo-harness-subagentes](https://github.com/betta-tech/ejemplo-harness-subagentes)** ⭐ 47 — 提供子代理（sub‑agents）协作模式的示例代码，帮助开发者快速理解 Agent 架构下的协调与执行流程。
